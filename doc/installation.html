Installation

LuaJava was tested with Windows and Linux platafforms using Java JDK 1.3 and 1.4. LuaJava is JDK 1.3 or higher compatible.

Lua Java source distribution came with a Windows nmakefile and a Linux makefile. Edit the makefile to point to your JDK, Lua directories and C libraries and compilers.

Windows Instructions

If you have nmake just type :

nmake -f nmakefile

Microsoft Compiler

If you use MSVC++ run this command before running nmake:

your_path_to_msvc\bin\VCVARS32.BAT

Example : 

  c:\luajava-1.0b4>"C:\Program Files\Microsoft Visual Studio\VC98\Bin\VCVARS32.bat"
  c:\luajava-1.0b4>nmake -f nmakefile

Linux Instructions

Just type make on your favorite shell.


Running the Lua Java Console

Lua Java came with a simple console. To run it type : 

 c:\luajava-1.0b4>java -cp "luajava-1.0b4.jar" luajava.Console
