Visão geral

O LuaJava é uma ferramenta de scripts para Java. O objeto da ferramenta é permitir que os scripts escritos em Lua manipulem componentes desenvolvidos em Java. O LuaJava permite o acesso aos componentes Java a partir de Lua, usando-se a mesma sintaxe utilizada para acessar os objetos nativos de Lua, sem a necessidade de fazer declarações ou qualquer tipo de pré-processamento.

O LuaJava também permite que o Java implemente uma interface usando Lua. Dessa forma, qualquer interface pode ser implementada em Lua e passada como parâmetro para qualquer método e, quando chamada, a função equivalente será chamada em Lua e o resultado da mesma, retornado ao Java.

O LuaJava está disponível na mesma licença do Lua 5.0, ou seja, ele pode ser usado sem qualquer custo, para fins acadêmicos ou comerciais.

Status

A versão atual é a 1,0. Ela foi desenvolvida para o Lua 5.0.

Download

O download do LuaJava pode ser feito na página LuaForge. Se você estiver usando LuaBinaries Release 2, uma versão do LuaJava pré-compilada para Windows também estará disponível na mesma página LuaForge.

Créditos

LuaJava 1.0
O LuaJava 1.0 foi desenvolvido por Thiago Ponte e foi patrocinado pela Ideais, pela Fábrica Digital, pela Finep e pelo CNPq.
LuaJava 1.0 beta
O LuaJava 1.0 beta foi reimplementado para ser compatível com o Lua 5.0 e com os novos recursos do Java 1.3. O LuaJava 1.0 beta foi desenvolvido por Thiago Ponte e Rafael Rizzato, com contribuições de João Machado, Danilo Tuler e Roberto Ierusalimschy.
LuaJava para Lua 3.1
O LuaJava foi desenvolvido originalmente por Carlos Cassino.
